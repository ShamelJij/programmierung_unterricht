Algorithmus

Ein Algorithmus ist eine definierte und endliche Vorgehensweise, mit der ein Problem gelöst werden.
Er besteht aus Anweisungen, welche Schritt für Schritt und in der definierten Reihenfolge abgearbeitet werden.
Ein Algorithmus ist erst einmal nichts weiter als eine Handlungsvorschrift, mit der man ein Problem lösen kann.
Algorithmen werden dann mithilfe von Programmen auf IT-Systemen umgesetzt. Dazu bedient man sich einer Programmiersprache, in welcher das Programm geschrieben wird.
Der Algorithmus selbst ist aber programmiersprachenunabhängig!

Python

        def ggt(a, b):
            if a == 0:
                return b
            while b != 0:
                if a > b:
                    a = a - b
                else:
                    b = b - a
            return a

Programm

"Programm" und "Sofware" sind Synonyme.
Es gibt jedoch einige Unterschiede!
Unter einem Programm kann man einen Text (Quellcode) verstehen, der aus Befehlen und Speicheradressen, in welchen Daten gespeichert werden, besteht.
Ein Compiler übersetzt diesen Text, welcher in einer bestimmten Programmiersprache verfasst ist, in Maschnenbefehle.
